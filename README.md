UfsConnectBook is a full-stack web application developed to modernize resource management at the University of the Free State. 
The system provides a centralized platform for students, staff, and guests to view real-time availability and reserve campus facilities (gyms, labs, laundry, and parking).
By transitioning from manual queues to a digital solution, this project aims to improve student productivity and optimize university infrastructure usage.
